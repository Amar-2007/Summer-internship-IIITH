Task5-- Video stacking
This task processes a traffic video using a YOLOv8 object detection model and then compares the original video with the detected output by stacking them vertically into a single video with audio.

traffic.mp4       → original input video
output.mp4        → YOLO detected output video
audio.mp3         → audio for the stack 
final_video.mp4 → final combined video with audio

Output:
Top: Original traffic video
Bottom: YOLO detection video
Final file: final_stacked.mp4
